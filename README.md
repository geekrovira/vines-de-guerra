# ✈️ Vines de Guerra — Aviones Militares Modernos

**40 aeronaves de combate que marcaron la historia de la aviación militar.**

Página web interactiva con perfil detallado de cazas stealth, bombarderos, helicópteros de ataque, drones, aviones de transporte y leyendas de la aviación.

---

## ✈️ Contenido

40 perfiles cubriendo:

- **Cazas 5ta Gen** — F-22 Raptor, F-35 Lightning II, Su-57 Felon, J-20 Chengdu
- **Cazas 4ta/4.5 Gen** — F-16, Su-35, F-15EX, Eurofighter Typhoon, Rafale, Gripen
- **Bombarderos** — B-2 Spirit, B-52 Stratofortress, B-21 Raider
- **Ataque** — A-10 Warthog, Su-25 Frogfoot
- **Helicópteros** — AH-64 Apache, Ka-52 Alligator, Mi-28 Havoc, CH-47 Chinook
- **Drones** — MQ-9 Reaper, Bayraktar TB2, RQ-4 Global Hawk
- **Leyendas** — SR-71 Blackbird, F-117 Nighthawk, F-14 Tomcat

Cada perfil incluye: historia completa, ficha técnica, impacto histórico.

## 🛠️ Tech

- HTML5 + CSS3 + JavaScript vanilla
- Filtros por generación (5ta Gen, 4ta Gen, Stealth, Ataque, Drones, Leyendas)
- Vista grid / lista
- Typewriter effect en modal
- Glitch effect con temática militar
- Easter eggs: Konami code, botón oculto
- Sin dependencias

```bash
git clone https://github.com/geekrovira/vines-de-guerra.git
cd vines-de-guerra
# Abre index.html
```
